if_

摘要

template< 
      typename C
    , typename T1
    , typename T2
    >
struct if_
{
    typedef unspecified type;
};

描述

根据 C 的值返回两个参数 T1 或 T2 中的一个。

头文件

#include <boost/mpl/if.hpp>

参数

参数 要求 描述
C 整型常量 选择的条件
T1, T2 任意类型 从中进行选择的类型

表达式语义

对于任意 整型常量 c 和任意类型 t1, t2:

typedef if_<c,t1,t2>::type t;
返回类型: 任意类型
语义: 如果 c::value == true, t 等同于 t1; 否则 t 等同于 t2.

例子

typedef if_<true_,char,long>::type t1;
typedef if_<false_,char,long>::type t2;

BOOST_MPL_ASSERT(( is_same<t1, char> ));
BOOST_MPL_ASSERT(( is_same<t2, long> ));

参见

元函数, 整型常量, if_c, eval_if
