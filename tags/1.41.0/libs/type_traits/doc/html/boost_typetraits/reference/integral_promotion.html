integral_promotion 

template <class T>
struct integral_promotion
{
   typedef see-below type;
};

类型：如果可以对类型 T 的一个右值进行整型提升，则对 T 进行整型提升并保持 T 的cv限定符，否则保持 T 不变。

C++ 标准参考：4.5 除了 4.5/3 (整数的位域)。

头文件： #include <boost/type_traits/integral_promotion.hpp> 或 #include <boost/type_traits.hpp>
