平凡元函数

描述

平凡元函数 以类类型 x 为单个参数，返回 x 的内嵌类型成员 x::name, 其中 name 为表示由特定元函数实例访问的实际成员名的占位符号。作为惯例，MPL中的所有 平凡元函数 以它们提供访问的成员名为名字。例如，名为 first 的 平凡元函数 访问 x 的内嵌成员 ::first.

表达式要求

在下表以及后续说明中，name 是表示 平凡元函数 本身以及被访问成员的名字的占位符号，而 x 是一个类类型，满足 x::name 是一个有效的 类型名。

表达式 类型 复杂度
name<x>::type 任意类型 常量时间

表达式语义

typedef name<x>::type r;
前置条件: x::name 是一个有效的 类型名
语义: is_same<r,x::name>::value == true.

Models

  • first
  • second
  • base

参见

元函数, 平凡元函数, identity
