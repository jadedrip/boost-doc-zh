is_signed 

template <class T>
struct is_signed : public true_type-or-false_type {};

继承：如果 T 是一个有符号整数类型或是一个底层为有符号整数类型的枚举类型，则从 true_type 继承，否则从 false_type 继承。

C++ 标准参考：3.9.1, 7.2.

头文件：#include <boost/type_traits/is_signed.hpp> 或 #include <boost/type_traits.hpp>

示例：

is_signed<int> 从 true_type 继承。

is_signed<int const volatile>::type 是 true_type 类型。

is_signed<unsigned int>::value 是一个求值为 false 的 integral constant expression（整常表达式）。

is_signed<myclass>::value 是一个求值为 false 的 integral constant expression（整常表达式）。

is_signed<char>::value 是一个值依赖于类型 char 的符号性的 integral constant expression（整常表达式）。

is_signed<long long>::value 是一个求值为 true 的 integral constant expression（整常表达式）。

is_signed<T>::value_type 是 bool 类型。
