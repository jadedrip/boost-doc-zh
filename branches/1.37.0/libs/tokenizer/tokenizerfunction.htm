TokenizerFunction 概念

TokenizerFunction 是一个函数对象，其目的是分析一个给定的序列，直至找到一个单词或到达序列末尾。然后它更新该单词，并通知调用者当前单词之后的下一步字符的位置。

强化自

Assignable可赋值的，CopyConstructable可复制构造的

符号

X 一个符合 TokenizerFunction 的类型
func 类型 X 的对象
tok 单词对象
next 指向被分析序列中第一个未分析字符的迭代器
end 指向被分析序列末尾的迭代器

定义

单词是指对序列进行分析所得到的结果。

有效表达式

除了可赋值和可复制构造所要求的表达式以外，以下表达式是有效的

名字 表达式 返回类型
Functor func(next, end, tok) bool
reset reset() void

表达式语义

除了可赋值和可复制构造的表达式语义以外，TokenizerFunction 还具有以下表达式语义

名字 表达式 前提条件 语义 后续条件
operator() func(next, end, tok) next 和 end 是同一序列的有效迭代器。next 是函数可自由修改的引用。tok 是已构造的。 返回值表示是否在序列 (next,end] 中找到新单词 如果返回值为 true, 则新单词被赋给 tok. next 总是被更新为下一次调用时开始分析的位置。
reset reset() 无 清除对象中分析当前序列所用的所有状态变量。 可以指定一个新的序列进行分析。

复杂度保证

无保证。TokenizerFunction 的 model 可以自由定义复杂度。
