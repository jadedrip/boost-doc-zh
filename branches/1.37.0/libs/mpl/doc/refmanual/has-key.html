has_key

摘要

template<
      typename Sequence
    , typename Key
    >
struct has_key
{
    typedef unspecified type;
};

描述

如果 Sequence 包含键值为 Key 的元素，则返回一个真值的 整型常量.

头文件

#include <boost/mpl/has_key.hpp>

Model of

Tag 分派元函数

参数

参数 要求 描述
Sequence 关联序列 被查询的序列
Key 任意类型 查询的键值

表达式语义

对于任意 关联序列 s, 以及任意类型 key:

typedef has_key<s,key>::type c;
返回类型: 布尔 整型常量
语义: c::value == true 如果 key 是 s 中的键值; 否则 c::value == false.

复杂度

分期常量时间。

例子

typedef map< pair<int,unsigned>, pair<char,long> > m;
BOOST_MPL_ASSERT_NOT(( has_key<m,long> ));

typedef insert< m, pair<long,unsigned long> > m1;
BOOST_MPL_ASSERT(( has_key<m1,long> ));

参见

关联序列, count, insert, erase_key
