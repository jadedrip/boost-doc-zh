Boost 随机数库

很多应用都需要随机数。使用 Boost 随机数库 (Boost.Random) 的生成器和分布模型可以产生多种具有有用性质的随机数 (例如正态分布)。

请先阅读 概念文档，熟悉一下基本概念。如果你想要尽快开始，不妨看看 random_demo.cpp。

下面提供一个例子：

  boost::mt19937 rng;                 // produces randomness out of thin air
                                      // see pseudo-random number generators
  boost::uniform_int<> six(1,6)       // distribution that maps to 1..6
                                      // see random number distributions
  boost::variate_generator<boost::mt19937&, boost::uniform_int<> >
           die(rng, six);             // glues randomness with mapping
  int x = die();                      // simulate rolling a die

库的组织

本库被分成了数个头文件，全部存放在 boost/random/。此外，boost/random.hpp 包含了 boost/random/ 中的所有头文件。

本库提供了一个前端类模板，variate_generate；请阅读其 文档。

  • boost/random/variate_generator.hpp
以下头文件中提供了数个随机数生成器，请阅读其 文档。
  • boost/random/linear_congruential.hpp
  • boost/random/additive_combine.hpp
  • boost/random/inversive_congruential.hpp
  • boost/random/shuffle_output.hpp
  • boost/random/mersenne_twister.hpp
  • boost/random/lagged_fibonacci.hpp
类似地，以下头文件中提供了数个随机数分布模型；请阅读其 文档。
  • boost/random/uniform_smallint.hpp
  • boost/random/uniform_int.hpp
  • boost/random/uniform_01.hpp
  • boost/random/uniform_real.hpp
  • boost/random/triangle_distribution.hpp
  • boost/random/bernoulli_distribution.hpp
  • boost/random/cauchy_distribution.hpp
  • boost/random/exponential_distribution.hpp
  • boost/random/geometric_distribution.hpp
  • boost/random/normal_distribution.hpp
  • boost/random/lognormal_distribution.hpp
  • boost/random/uniform_on_sphere.hpp
此外，头文件 <boost/nondet_random.hpp> 中提供了不确定的随机数生成器；请阅读其 文档。

为把生成器界面和分布函数映射到其它概念##上，提供了数个 修饰符。

理论注记

用于产生确定的和不确定的随机数的方法有根本的不同。由于计算机确定的本性，实现不确定的随机数生成设施往往很困难。因此，随机数库被分成了两部分，以反映两个不同的应用领域。

历史和致谢

1999 年 11 月，Jeet Sukumaran 提出了一个基于虚函数的框架，不久做出了一个基于模板的原型。Ed Brey 指出 Microsoft Visual C++ 不支持类内成员初始化，并建议采用 enum 来绕过这个问题。Dave Abrahams 指出了离散化问题。

2000 年 3 月，经过 boost 邮件列表的多方面讨论，决定首次释出本随机数库。感谢 Beman Dawes 提供最初的 min_rand 类、可移植性修正、文档建议和一般的指导。Harry Erwin 提供的头文件揭示出了更多的需求。Ed Brey 和 Beman Dawes 提出需要一个类迭代器的界面。

Beman Dawes 组织了正式审核。其间 Matthias Troyer, Csaba Szepesvari 和 Thomas Holenstein 提供了详尽的评论。审核通过的版本在 2000 年 6 月 17 日正式成为了 boost 的一部分。

Gary Powell 贡献了关于代码清洁性的建议。Dave Abrahams 和 Howard Hinnant 建议把基本的生成器模板从名字空间 boost::detail 移至 boost::random。

Ed Brey 要求移除了多余的警告并帮助解决了 uint64_t 处理问题。Andreas Scherer 在 MSVC 上测试了本库。 Matthias Troyer 贡献了一个 lagged-Fibonacci 算法的生成器。Michael Stevens 发现了 normal_distribution 复制语义中的 bug，并对文档改进提出了若干建议。
