value_type

摘要

template<
      typename Sequence
    , typename X
    >
struct value_type
{
    typedef unspecified type;
};

描述

返回在 Sequence 中用于元素 X 的 值。

头文件

#include <boost/mpl/value_type.hpp>

Model of

Tag 分派元函数

参数

参数 要求 描述
Sequence 关联序列 被查询的序列
X 任意类型 获取 值 的类型

表达式语义

对于任意 关联序列 s, 以及任意类型 x:

typedef value_type<s,x>::type v;
返回类型: 一个类型
前置条件: x 可被放入 s.
语义: v 是在 s 中用于 x 的 值
后置条件:

如果 .. parsed-literal:

has_key< s,key_type<s,x>::type >::type

则 .. parsed-literal:

at< s,key_type<s,x>::type >::type

等同于 value_type<s,x>::type.

复杂度

分期常量时间。

例子

typedef value_type< map<>,pair<int,unsigned> >::type v1;
typedef value_type< set<>,pair<int,unsigned> >::type v2;

BOOST_MPL_ASSERT(( is_same< v1,unsigned > ));
BOOST_MPL_ASSERT(( is_same< v2,pair<int,unsigned> > ));

参见

关联序列, key_type, at, set, map
